# Task System Overview

The Systemprompt Coding Agent is an AI-powered development assistant that enables automated code generation and modification through a unique three-tier architecture. The system allows AI agents (Claude or Gemini) to work directly on project files while maintaining git branch isolation for safe experimentation. At its core, it uses a Docker container running an MCP (Model Context Protocol) server for client communication, a host-based daemon that bridges between the containerized server and the actual filesystem, and AI agent processes that execute on the host machine with full access to the project repository. This design ensures that AI agents can perform real git operations, create and modify files, and execute commands in the actual development environment, while the Docker container provides a consistent and isolated protocol server interface.
